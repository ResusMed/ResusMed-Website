# ResusMed Leads Plugin

Custom WordPress plugin for managing ResusMed student and corporate training leads.

## Purpose

Tracks leads, course interests, follow-up dates, email sequences, notes, lead status, and lead activity.

## Main Features

- Add and manage leads
- Track TNCC and ENPC interest
- Send 3-step email sequences
- Track sent/opened/clicked/bounced where supported
- Manage follow-up date/time
- Store timestamped notes
- Mark Active, Inactive, Converted, DNC
- Manual email template sending

## Do Not Store Here

- Student data
- WooCommerce exports
- API keys
- wp-config.php
- SendLayer credentials
- ClickSend credentials
- Database backups
- PHI or private customer information

## Deployment Rule

No code should go live unless:
1. Version number is updated.
2. CHANGELOG.md is updated.
3. Plugin ZIP is tested on staging.
4. Live rollback version is known.
